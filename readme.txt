This is a project I completed as an undergraduate. I commited the code as-is but had to create a visual studio project.

The testing program accepts command line arguments [AI Name 1] [AI Name 2] [Depth 1] [Depth 2] [Number of Games (optional)]. 
If depth does not apply to the AI you are using then you can enter any number.

To play a game against the computer try the following arguments:
MancalaGameEngineAndAI.exe human myai 1 5

To play a game against the computer try the following arguments and then open log.dat to see a summary of the results:
MancalaGameEngineAndAI.exe random myai 1 5 100
